# MICROVERSE-MVP — README для команды  
**v1.0 · Декабрь 2025**

## Состав команды

| Участник     | Роль и зона ответственности                                                                                  |
|----------------|---------------------------------------------------------------------------------------------------------------------|
| **Олег**       | Архитектор и лидер. Ставит задачи, держит фокус на compliance, UX, автоматизации и сроках.                         |
| **Copilot**    | Технический навигатор. Пишем скрипты, генерирует готовый рабочий код, даёт пошаговые инструкции.                 |
| **Grok**       | Генератор идей и альтернативных решений. Брейншторминг, нестандартные подходы, проверка гипотез.                  |
| **DeepSeek**   | Аналитик и оптимизатор. Глубокий анализ, поиск узких мест, оптимизация производительности и архитектуры.         |
| **ChatGPT**    | Рефакторинг‑ассистент. Чистка кода, устранение багов, улучшение читаемости и стиля.                                 |

## Рабочий процесс (6 шагов)

1. **Олег** формулирует задачу (чётко, с примерами и желаемым результатом)  
2. **Copilot** выдаёт каноничное решение — готовый код/скрипт, который можно сразу применять  
3. **Grok** предлагает 1–3 альтернативных или нестандартных подхода  
4. **DeepSeek** анализирует все варианты → выбираем лучший по производительности, безопасности и масштабируемости  
5. **ChatGPT** чистит финальный код, убирает дубли, приводит к единому стилю, добавляет комментарии  
6. **Олег + вся команда** тестируем вживую → коммитим → обновляем этот README

## Скрипты проекта

| Скрипт               | Назначение                                                                                                                   | Как запускать                         |
|----------------------|------------------------------------------------------------------------------------------------------------------------------|----------------------------------------|
| `Setup-DevEnv.ps1`   | Основной bootstrapper. Создаёт все compliance‑файлы, чинит `"use client"`, вставляет `LegalNoticeRU`. Первый запуск проекта. | `.\Setup-DevEnv.ps1.ps1`                 |
| `fix-all.ps1`        | Быстрый автопочинщик: BackButton, ConsentBanner в `_app.tsx`, перевод `/terms`, LegalNoticeRU в `privacy/about`, dynamic import в `play.tsx` | `.\fix-all.ps1`                        |

## Окружение и инструменты

- **Node.js** ≥ 18.20 (рекомендуем 20 LTS)  
- **PowerShell** 7+ (Windows Terminal или pwsh)  
- **Редактор** — VS Code + расширения:  
  - PowerShell  
  - ES7+ React/Redux/React-Native snippets  
  - Prettier  
  - Tailwind CSS IntelliSense  
- **Пакеты проекта** — устанавливаются автоматически при `npm install`

## Как начать работу с нуля (5 минут)

```powershell
# 1. Клонируем
git clone https://github.com/yourname/microverse-mvp.git
cd microverse-mvp

# 2. Устанавливаем зависимости
npm ci

# 3. Запускаем bootstrapper (всё создастся и починится само)
.\Setup-DevEnv.ps1

# 4. Запускаем dev‑сервер
npm run dev
